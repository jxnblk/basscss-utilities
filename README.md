# BASSCSS Utilities

Basic display, layout, typographic, and white space utilities for BASSCSS for use with Rework.

## Configurable Variables

- `--space-1`
- `--space-2`
- `--space-3`
- `--space-4`

## Included Classes

- inline
- block
- inline-block

- overflow-hidden
- left
- right
- clearfix
- fit

- bold
- regular
- medium
- italic
- caps

- left-align
- center
- right-align
- justified
- nowrap

Plus white space scale
